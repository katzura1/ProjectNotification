Membuat pengingat obat

Berikut kegunaan masing-masing dari menu:

1.      Tombol Tambah : untuk membuat alarm baru anda dapat menggunakan menu ini, dengan cara mengklik tombol tersebut, maka akan diarahkan ke tampilan untuk input data alarm.

2.      Menu Histori : menu ini berguna untuk melihat histori data pengingat.

 

Langkah-langkah membuat pengingat:

1.      Klik tombol tambah, maka akan diarahkan ke formulir input data pengingat.

2.      Pada nama pengingat, klik untuk mengisi, akan muncul pilihan jenis obat yang dimakan yaitu, obat pagi, obat siang, obat malam.

3.      Selanjutnya, untuk nama obat silahkan isi, tersedia dua kolom, apabila hanya memakan satu obat, maka isi kolom yang pertama saja.

4.      Pada tanggal klik untuk mengganti tanggal, pilih tanggal sesuai dengan kapan pengingat akan muncul.

5.      Pada Jam klik untuk mengganti jam, pilih pada pukul dan menit berapa pengingat akan muncul.

6.      Tekan tombol centang untuk menyimpan pengingat yang telah diisi, dan apabila batal menyimpan tekan tombol silang yang berada di kiri sudut atas.

7.      Maka pengingat yang sudah dibuat akan muncul pada tampilan menu alarm.
